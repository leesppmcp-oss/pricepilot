# **App Name**: PricePilot

## Core Features:

- Product Input: Allows users to enter the name of the product they want to compare prices for.
- Website Selection: Provides a list of supported websites (e.g., PChome, Momo, Yahoo, eBay) for price comparison.
- Price Fetcher: Queries the selected websites to programmatically locate the current price of the user-submitted product.
- Price Comparison Table: Displays a table comparing the prices of the product across all selected websites, indicating the lowest price.
- AI-Powered Deal Finder: A tool using a generative AI to intelligently identify and present related information to potential money saving offers from each website such as coupon codes, bundle deals, or discounted shipping.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to convey trust and reliability.
- Background color: A light gray (#F0F0F0) for a clean and neutral backdrop.
- Accent color: An energetic orange (#FF8C00) to highlight the best deals and CTAs.
- Body and headline font: 'Inter' for a clean, modern, and readable experience.
- Simple, clear icons for product categories and website logos.
- Clean and organized layout with a focus on the price comparison table.
- Subtle transitions and loading animations to enhance user experience.
